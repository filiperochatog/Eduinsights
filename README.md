# EduInsights 📊📚

EduInsights é uma plataforma de visualização de dados e de simulados, projetada para fornecer insights educacionais de maneira intuitiva e interativa.

## Tecnologias Utilizadas

- **Frontend:** 🖥️
  - HTML
  - CSS
  - JavaScript
  - Chart.js

- **Backend:** 🔧
  - Django
  - Django REST Framework

- **Banco de Dados:** 🗄️
  - SQLite
